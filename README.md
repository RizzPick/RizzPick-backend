# RizzPick

## {서비스 사진 예정}

## 프로젝트 소개

> "우리 뭐할까요?"라는 질문에서 출발해 사용자들이 매력적인 데이트 계획을 통해 깊고 의미 있는 관계를 맺을 수 있도록 유도하고자 합니다.
---
## ✅ 서비스 핵심 기능

### **1. 데이트 계획 공유 및 선택** 
>사용자들이 개인적으로 기획한 데이트 아이디어를 공유하고, 상대방의 계획을 보며 선택하는 기능입니다.

### **2. 개인 프로필 기반 추천**

> 지역, 성별을 기반으로 상대방과의 추천이 이루어집니다.
